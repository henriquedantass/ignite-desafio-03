## Iniciando LocalHost
Primeiro, inicie a fake api para ter acesso aos dados dos produtos;

```bash
yarn server
```

## Iniciando LocalHost

```bash
yarn start
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
